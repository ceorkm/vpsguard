vpsguard apt repository

Install the agent on Debian or Ubuntu:

curl -fsSL https://ceorkm.github.io/vpsguard/apt/vpsguard.asc | sudo tee /etc/apt/keyrings/vpsguard.asc > /dev/null
echo "deb [signed-by=/etc/apt/keyrings/vpsguard.asc] https://ceorkm.github.io/vpsguard/apt stable main" | sudo tee /etc/apt/sources.list.d/vpsguard.list
sudo apt update
sudo apt install vpsguard
sudo vpsguard configure

Source: github.com/ceorkm/vpsguard
